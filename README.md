## Тестовое задание Облако Group frontend часть 

## Загрузка репозитория
```bash
$ git clone git@github.com:UtopicUnicorn/test-task-CloudGroup-frontend.git
```
## Переход в директорю с проектом
```bash
$ cd test-task-CloudGroup-frontend
```

## Установка
```bash
$ npm install
```

## Запуск приложения

```bash
$ ng serve
```
